# Hourly Research Automation

This folder configures an automatic research loop for the project.

## What it does

- Runs every 1 hour.
- Each run is limited to 15 minutes.
- Searches GitHub for Unity/card-game related repos and tech.
- Appends findings to:
  - `Temp/research/hourly_research_log.md`
  - `Temp/research/latest_candidates.json`
  - `Temp/research/implementation_queue.md`

## Scripts

- `Tools/Research/HourlyResearch.ps1`
- `Tools/Research/SetupHourlyResearchTask.ps1`
- `C:/BaralhoResearch/RunHourlyResearch.cmd` (generated by setup script)

## Task name

- `BaralhoPrototypeHourlyResearch`

## Install / reinstall task

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File Tools/Research/SetupHourlyResearchTask.ps1
```

## Run once manually

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File Tools/Research/HourlyResearch.ps1 -DurationMinutes 15
```
